Reloj analógico con manecillas de hora, minuto y segundo (movimiento suave gracias a getMilliseconds()). Incluye display digital de la hora formateada.

Cómo funciona

Se usa new Date() para obtener hora, minuto, segundo y milisegundo actuales. Se limpia el canvas y se redibuja el círculo del reloj con marcas de hora (12 líneas). Cada manecilla se dibuja usando ctx.save(), ctx.translate(cx, cy), ctx.rotate(angle), fillRect() y ctx.restore().

function drawClock() {
  var now = new Date();
  var sec = now.getSeconds() + now.getMilliseconds() / 1000;
  var min = now.getMinutes() + sec / 60;
  var hr = now.getHours() % 12 + min / 60;

  ctx.clearRect(0, 0, canvas.width, canvas.height);
  /* círculo exterior, marcas de hora */

  // Manecilla de hora
  ctx.save();
  ctx.translate(cx, cy);
  ctx.rotate(hr * (Math.PI / 6));
  ctx.fillRect(-3, -halfSize * 0.5, 6, halfSize * 0.5);
  ctx.restore();

  // Manecilla de minuto
  ctx.save();
  ctx.translate(cx, cy);
  ctx.rotate(min * (Math.PI / 30));
  ctx.fillRect(-2, -halfSize * 0.75, 4, halfSize * 0.75);
  ctx.restore();

  // Manecilla de segundo
  ctx.save();
  ctx.translate(cx, cy);
  ctx.rotate(sec * (Math.PI / 30));
  ctx.fillRect(-1, -halfSize * 0.8, 2, halfSize * 0.8);
  ctx.restore();
}

Incluir getMilliseconds() en el cálculo de segundos evita el tic-tac discretoy produce un movimiento continuo y suave. El display digital se actualiza con String(hr).padStart(2, "0").

APIs de JavaScript utilizadas

  • Date — new Date(), getHours(), getMinutes(), getSeconds(), getMilliseconds()
  • CanvasRenderingContext2D — save()/restore(), translate(), rotate(), arc(), fillRect()
  • Math.PI — Conversión de ángulos: PI/6 (30° por hora), PI/30 (6° por minuto/segundo)
  • String.padStart() — Formateo del display digital con dos dígitos
  • requestAnimationFrame — Redibujado en cada frame (~60 fps) para movimiento suave
