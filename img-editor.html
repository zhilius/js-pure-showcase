Carga de imágenes por selección o drag & drop con 5 filtros: grayscale, sepia, invert, brightness y blur (promedio 3×3). Demuestra manipulación píxel a píxel con Uint8ClampedArray.

Cómo funciona

El usuario arrastra o selecciona una imagen. Se usa FileReader.readAsDataURL() para cargar la imagen en un Image element, luego se dibuja en un canvas oculto para extraer los píxeles con getImageData(). Cada filtro recorre el Uint8ClampedArray y modifica los valores RGBA:

function applyGrayscale(pixels) {
  for (var i = 0; i < pixels.length; i += 4) {
    var r = pixels[i], g = pixels[i + 1], b = pixels[i + 2];
    var gray = 0.299 * r + 0.587 * g + 0.114 * b;
    pixels[i] = pixels[i + 1] = pixels[i + 2] = gray;
  }
}

function applySepia(pixels) {
  for (var i = 0; i < pixels.length; i += 4) {
    var r = pixels[i], g = pixels[i + 1], b = pixels[i + 2];
    pixels[i] = Math.min(255, r * 0.393 + g * 0.769 + b * 0.189);
    pixels[i + 1] = Math.min(255, r * 0.349 + g * 0.686 + b * 0.168);
    pixels[i + 2] = Math.min(255, r * 0.272 + g * 0.534 + b * 0.131);
  }
}

function applyBlur(pixels, w, h) {
  var copy = new Uint8ClampedArray(pixels);
  for (var y = 1; y < h - 1; y++) {
    for (var x = 1; x < w - 1; x++) {
      var idx = (y * w + x) * 4;
      for (var c = 0; c < 3; c++) {
        pixels[idx + c] = (
          copy[idx - w*4 + c] + copy[idx + w*4 + c] +
          copy[idx - 4 + c] + copy[idx + 4 + c] +
          copy[idx + c]
        ) / 5;
      }
    }
  }
}

La carga por drag & drop usa dragenter/dragover/drop con e.dataTransfer.files[0] y e.preventDefault(). El formato de archivo se valida con file.type.startsWith("image/").

APIs de JavaScript utilizadas

  • File API — FileReader.readAsDataURL() para leer el archivo como URL de datos
  • Drag & Drop — dragenter, dragover, drop con dataTransfer.files
  • CanvasRenderingContext2D — drawImage() para renderizar la imagen, getImageData()/putImageData() para acceso a píxeles
  • Uint8ClampedArray — Arreglo tipado con clamping 0–255 para manipulación directa de RGBA
  • HTMLInputElement — input[type=file] con accept="image/*" y listener change
