Qué enseña

Canvas 2DEfecto visualAnimación por frame

Efecto de lluvia de caracteres katakana al estilo Matrix. Demuestra el uso de requestAnimationFrame con un canvas semitransparente para crear el efecto de estela.

Cómo funciona

Se define un arreglo de 30 columnas, cada una con un objeto que tiene y (posición vertical) y speed (velocidad en píxeles por frame). En cada frame se pinta un rectángulo negro semitransparente sobre todo el canvas (fillStyle = "rgba(0,0,0,0.05)"), creando el efecto de que los caracteres se desvanecen gradualmente.

var columns = 30;
var drops = [];
for (var i = 0; i < columns; i++) {
  drops[i] = { y: Math.random() * -h, speed: 1 + Math.random() * 3 };
}

function draw() {
  ctx.fillStyle = "rgba(0,0,0,0.05)";
  ctx.fillRect(0, 0, canvas.width, canvas.height);
  ctx.fillStyle = "#0f0";
  ctx.font = fontSize + "px monospace";

  for (var i = 0; i < drops.length; i++) {
    var char = String.fromCharCode(0x30A0 + Math.random() * 96);
    ctx.fillText(char, i * fontSize, drops[i].y);
    drops[i].y += drops[i].speed;
    if (drops[i].y > canvas.height) {
      drops[i].y = -fontSize;
      drops[i].speed = 1 + Math.random() * 3;
    }
  }
  requestAnimationFrame(draw);
}

Los caracteres se generan aleatoriamente del bloque Unicode Katakana (U+30A0–U+30FF). Cada gota se reinicia desde arriba con una velocidad aleatoria al salir de la pantalla.

APIs de JavaScript utilizadas

  • CanvasRenderingContext2D — fillText() para dibujar caracteres, fillRect() semitransparente para la estela
  • requestAnimationFrame — Loop de animación para el movimiento continuo
  • String.fromCharCode() — Generación de caracteres Unicode katakana aleatorios en el rango U+30A0–U+30FF
  • Math.random() — Posición inicial aleatoria, velocidad variable y caracteres aleatorios
