Dictado por voz (reconocimiento) y texto a voz (síntesis). Demuestra el uso de SpeechRecognition con resultados interinos y SpeechSynthesisUtterance con control de tono y velocidad.

Cómo funciona

Reconocimiento: Se instancia webkitSpeechRecognition (única implementación disponible en la mayoría de navegadores), se configura lang: "es-AR", continuous: true e interimResults: true. En onresult se itera event.results para construir el texto reconocido, diferenciando resultados finales (event.results[i].isFinal) de interinos.

var recognition = new webkitSpeechRecognition();
recognition.lang = "es-AR";
recognition.continuous = true;
recognition.interimResults = true;

recognition.onresult = function (event) {
  var transcript = "";
  for (var i = event.resultIndex; i < event.results.length; i++) {
    transcript += event.results[i][0].transcript;
  }
  textarea.value = transcript;
};

Síntesis: Se crea un SpeechSynthesisUtterance con el texto ingresado, se configura lang: "es-AR", rate y pitch según los sliders, y se pasa a speechSynthesis.speak(). Un botón detiene la síntesis con speechSynthesis.cancel().

APIs de JavaScript utilizadas

  • webkitSpeechRecognition — Reconocimiento de voz con onresult, onerror, continuous e interimResults
  • SpeechSynthesis — speechSynthesis.speak(), speechSynthesis.cancel(), speechSynthesis.paused
  • SpeechSynthesisUtterance — Configuración de lang, rate, pitch y text
  • DOM Events — click para botones de inicio/detención, input para sliders de tono y velocidad
