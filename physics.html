Qué enseña

Canvas 2DFísica clásicaAnimación por frame

Bolas con gravedad, rebote elástico, fricción y highlights especulares. Demuestra integración de física paso a paso en cada frame de animación.

Cómo funciona

Cada bola tiene posición (x, y), velocidad (vx, vy), radio, color, y coeficiente de restitución (e). En cada frame:

b.vy += grav;          // gravedad
b.x += b.vx; b.y += b.vy;  // mover

// Colisión contra bordes
if (b.x - b.r < 0) { b.x = b.r; b.vx *= -b.e; }
if (b.y + b.r > 180) {
  b.y = 180 - b.r;
  b.vy *= -b.e;       // rebote
  b.vx *= 0.98;       // fricción al tocar el suelo
}

El coeficiente de restitución e es aleatorio entre 0.7 y 0.9, simulando distintos materiales. El highlight especular se dibuja como un círculo blanco semitransparente desplazado hacia arriba a la izquierda del centro de cada bola.

APIs de JavaScript utilizadas

  • CanvasRenderingContext2D — arc() para círculos, fillStyle con transparencia para highlights
  • requestAnimationFrame — Loop de física sincronizado con el refresco del monitor
  • Math.random() — Generación de valores aleatorios para radio, velocidad y restitución
  • Input range — <input type="range"> para control de gravedad en tiempo real
