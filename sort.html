Visualización en vivo de Bubble Sort y Quick Sort side-by-side sobre el mismo arreglo de 60 elementos. Demuestra el uso de async/await con setTimeout para animar algoritmos paso a paso.

Cómo funciona

El arreglo se inicializa con valores 1..60 y se mezcla con Fisher-Yates. Cada algoritmo itera de forma asíncrona: en cada paso marca los elementos que está comparando (rojo), pausa con await new Promise(r => setTimeout(r, speed)), y redibuja el canvas. Los elementos ya ordenados se muestran en verde.

async function bubbleSort() {
  running = true;
  for (var i = 0; i < arr.length; i++) {
    for (var j = 0; j < arr.length - i - 1; j++) {
      comparing = [j, j + 1]; draw();
      await new Promise(function (r) { setTimeout(r, getSpeed()); });
      if (arr[j] > arr[j + 1]) { /* swap */ }
    }
  }
  running = false;
}

async function quickSort() {
  async function partition(lo, hi) { /* ... */ }
  async function sort(lo, hi) {
    if (lo >= hi || !running) return;
    var pi = await partition(lo, hi);
    await sort(lo, pi - 1);
    await sort(pi + 1, hi);
  }
  await sort(0, arr.length - 1);
}

Quick Sort implementa partición con pivote (último elemento del subarreglo) y recursión async. El toggle entre algoritmos reinicia el arreglo al mismo estado.

APIs de JavaScript utilizadas

  • Async/Await — async function + await para pausar la ejecución del algoritmo en cada paso
  • Promise — new Promise(resolve => setTimeout(resolve, ms)) como sleep
  • CanvasRenderingContext2D — fillRect() para dibujar barras de altura proporcional al valor
  • Fisher-Yates shuffle — Algoritmo O(n) para mezclar el arreglo inicial
