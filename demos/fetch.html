Llamada a https://jsonplaceholder.typicode.com/users con la Fetch API. Demuestra el patrón de 3 estados (loading, success, error) y el renderizado dinámico de datos en el DOM.

Cómo funciona

Al hacer click en el botón «Cargar usuarios», se muestra un spinner de carga. Luego se ejecuta fetch(url) que devuelve una Promise. Se encadenan .then(res => res.json()) para parsear la respuesta y .then(data => renderUsers(data)) para mostrar los datos. Si hay un error (red, HTTP no-ok, etc.), se captura con .catch() y se muestra un mensaje de error.

function loadUsers() {
  statusEl.textContent = 'Cargando...';
  dataEl.innerHTML = '';

  fetch('https://jsonplaceholder.typicode.com/users')
    .then(function (res) {
      if (!res.ok) throw new Error('HTTP ' + res.status);
      return res.json();
    })
    .then(function (users) {
      statusEl.textContent = users.length + ' usuarios cargados';
      users.forEach(function (u) {
        var card = document.createElement('div');
        card.className = 'user-card';
        card.innerHTML = '<strong>' + u.name + '</strong><br>' +
          '✉ ' + u.email + '<br>' +
          '📍 ' + u.address.city;
        dataEl.appendChild(card);
      });
    })
    .catch(function (err) {
      statusEl.textContent = 'Error: ' + err.message;
    });
}

Se verifica res.ok para detectar errores HTTP (4xx/5xx). Cada usuario se renderiza como una tarjeta con nombre, email y ciudad, insertada con appendChild.

APIs de JavaScript utilizadas

  • fetch() — API moderna para peticiones HTTP, devuelve una Promise
  • Response.json() — Parseo de la respuesta como JSON
  • Promise.then() / .catch() — Encadenamiento asíncrono para flujo de datos y manejo de errores
  • document.createElement() — Creación dinámica de elementos DOM para cada usuario
  • Element.innerHTML / appendChild() — Inserción de contenido renderizado en el DOM
