Load images by selection or drag & drop with 5 filters: grayscale, sepia, invert, brightness and blur (3×3 average). Demonstrates pixel-by-pixel manipulation with Uint8ClampedArray.

How It Works

The user drags or selects an image. FileReader.readAsDataURL() loads the image into an Image element, then it is drawn on a hidden canvas to extract pixels with getImageData(). Each filter iterates over the Uint8ClampedArray and modifies RGBA values:

function applyGrayscale(pixels) {
  for (var i = 0; i < pixels.length; i += 4) {
    var r = pixels[i], g = pixels[i + 1], b = pixels[i + 2];
    var gray = 0.299 * r + 0.587 * g + 0.114 * b;
    pixels[i] = pixels[i + 1] = pixels[i + 2] = gray;
  }
}

function applySepia(pixels) {
  for (var i = 0; i < pixels.length; i += 4) {
    var r = pixels[i], g = pixels[i + 1], b = pixels[i + 2];
    pixels[i] = Math.min(255, r * 0.393 + g * 0.769 + b * 0.189);
    pixels[i + 1] = Math.min(255, r * 0.349 + g * 0.686 + b * 0.168);
    pixels[i + 2] = Math.min(255, r * 0.272 + g * 0.534 + b * 0.131);
  }
}

function applyBlur(pixels, w, h) {
  var copy = new Uint8ClampedArray(pixels);
  for (var y = 1; y < h - 1; y++) {
    for (var x = 1; x < w - 1; x++) {
      var idx = (y * w + x) * 4;
      for (var c = 0; c < 3; c++) {
        pixels[idx + c] = (
          copy[idx - w*4 + c] + copy[idx + w*4 + c] +
          copy[idx - 4 + c] + copy[idx + 4 + c] +
          copy[idx + c]
        ) / 5;
      }
    }
  }
}

Drag & drop loading uses dragenter/dragover/drop with e.dataTransfer.files[0] and e.preventDefault(). The file format is validated with file.type.startsWith("image/").

JavaScript APIs Used

  • File API — FileReader.readAsDataURL() to read the file as a data URL
  • Drag & Drop — dragenter, dragover, drop with dataTransfer.files
  • CanvasRenderingContext2D — drawImage() to render the image, getImageData()/putImageData() for pixel access
  • Uint8ClampedArray — Typed array with 0–255 clamping for direct RGBA manipulation
  • HTMLInputElement — input[type=file] with accept="image/*" and change listener
