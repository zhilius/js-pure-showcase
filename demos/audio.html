What It Teaches

Web Audio APIgetUserMediaAnalyserNodeOscillatorNodeGainNodeUint8Array

Two modes: (1) real microphone via getUserMedia() connected to an AnalyserNode that provides frequency data, and (2) synthesizer that generates 4 musical notes with oscillators.

How It Works

Microphone mode: Gets the audio stream with getUserMedia({ audio: true }), creates an AudioContext, connects the stream to an AnalyserNode with fftSize: 128. On each frame it reads the frequency data with getByteFrequencyData() as a Uint8Array and draws colored bars with hsl() on the canvas.

var stream = await navigator.mediaDevices.getUserMedia({ audio: true });
var audioCtx = new AudioContext();
var src = audioCtx.createMediaStreamSource(stream);
var analyser = audioCtx.createAnalyser();
analyser.fftSize = 128;
src.connect(analyser);
dataArr = new Uint8Array(analyser.frequencyBinCount);

function loop() {
  analyser.getByteFrequencyData(dataArr);
  /* draw bars according to dataArr[i] / 255 * 140 */
  requestAnimationFrame(loop);
}

Synthesizer mode: Creates 4 oscillators with frequencies of C, E, G, C (261.63, 329.63, 392, 523.25 Hz). Each has a GainNode with immediate attack and exponential decay over 0.4 seconds, creating an arpeggio.

JavaScript APIs Used

  • MediaDevices.getUserMedia() — Access to the user's microphone
  • AudioContext — Audio context for processing and generating sound
  • AnalyserNode — Real-time frequency analysis with FFT
  • OscillatorNode — Sine wave tone generation with configurable frequency
  • GainNode — Volume control with setValueAtTime and exponentialRampToValueAtTime
  • Uint8Array — Typed buffer for analyzer frequency data
