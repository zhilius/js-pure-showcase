Web Speech APISpeechRecognitionSpeechSynthesiswebkit prefix

Voice dictation (recognition) and text-to-speech (synthesis). Demonstrates using SpeechRecognition with interim results and SpeechSynthesisUtterance with pitch and rate control.

How It Works

Recognition: webkitSpeechRecognition is instantiated (the only implementation available in most browsers), configured with lang: "es-AR", continuous: true and interimResults: true. In onresult, event.results is iterated to build the recognized text, differentiating final results (event.results[i].isFinal) from interim ones.

var recognition = new webkitSpeechRecognition();
recognition.lang = "es-AR";
recognition.continuous = true;
recognition.interimResults = true;

recognition.onresult = function (event) {
  var transcript = "";
  for (var i = event.resultIndex; i < event.results.length; i++) {
    transcript += event.results[i][0].transcript;
  }
  textarea.value = transcript;
};

Synthesis: A SpeechSynthesisUtterance is created with the input text, configured with lang: "es-AR", rate and pitch from the sliders, and passed to speechSynthesis.speak(). A button stops synthesis with speechSynthesis.cancel().

JavaScript APIs Used

  • webkitSpeechRecognition — Voice recognition with onresult, onerror, continuous and interimResults
  • SpeechSynthesis — speechSynthesis.speak(), speechSynthesis.cancel(), speechSynthesis.paused
  • SpeechSynthesisUtterance — Configuration of lang, rate, pitch and text
  • DOM Events — click for start/stop buttons, input for pitch and rate sliders
