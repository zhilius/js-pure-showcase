What It Teaches

Canvas 2DrequestAnimationFrameMouse EventsTouch EventsDistance Detection

80 particles that repel from the cursor and draw connections between each other when close. Demonstrates efficient rendering with requestAnimationFrame, distance detection with Math.sqrt, relative coordinates with getBoundingClientRect(), and touch support with touchmove.

How It Works

Each particle has a position (x, y), velocity (vx, vy), radius, and color. On each frame:

for (var i = 0; i < particles.length; i++) {
  var p = particles[i];
  var dx = mouse.x - p.x, dy = mouse.y - p.y;
  var dist = Math.sqrt(dx * dx + dy * dy);
  if (dist < 60 && dist > 0) {
    var force = (60 - dist) / 60;
    p.vx -= (dx / dist) * force * 0.8;
    p.vy -= (dy / dist) * force * 0.8;
  }
  p.vx *= 0.96; p.vy *= 0.96;
  p.x += p.vx; p.y += p.vy;
  /* draw connections with nearby neighbors */
}

If the distance to the mouse is less than 60px, the particle is pushed in the opposite direction with a force inversely proportional to the distance. Friction (×0.96) gradually slows the movement. Lines are drawn between particles if they are within 60px of each other.

JavaScript APIs Used

  • CanvasRenderingContext2D — arc(), fillRect(), moveTo()/lineTo() for drawing particles and connections
  • requestAnimationFrame — Animation loop synchronized with the display refresh rate
  • MouseEvent — mousemove and mouseleave with getBoundingClientRect() for relative coordinates
  • TouchEvent — touchmove and touchend with e.touches[0] for mobile support
  • Math — Math.sqrt(), Math.random(), Math.sin() for movement and colors
