Functional calculator with addition, subtraction, multiplication and division. Demonstrates delegated event handling, number formatting, division-by-zero detection, and 12-digit limit.

How It Works

Two state variables are used: currentValue (the number being typed) and operator (the pending operation). When a digit is pressed it is concatenated to currentValue. When an operator is pressed, the current value is stored as storedValue and currentValue is cleared. When = is pressed, the stored operation is executed on storedValue and currentValue.

function handleClick(e) {
  var val = e.target.textContent;
  if (!isNaN(val) || val === '.') {
    if (currentValue.length < 12) currentValue += val;
  } else if (['+', '-', '×', '÷'].indexOf(val) >= 0) {
    if (operator) currentValue = compute();
    storedValue = currentValue;
    operator = val;
    currentValue = '';
  } else if (val === '=') {
    currentValue = compute();
    operator = '';
  }
  display.textContent = currentValue || '0';
}

function compute() {
  var a = parseFloat(storedValue), b = parseFloat(currentValue);
  if (operator === '÷' && b === 0) return 'Error';
  var result = {
    '+': a + b, '-': a - b, '×': a * b, '÷': a / b
  }[operator];
  return String(parseFloat(result.toFixed(10)));
}

The C button resets all values. Division by zero shows "Error".

JavaScript APIs Used

  • Event delegation — A single click listener on the button container using e.target.textContent
  • parseFloat() — Conversion from string to number for arithmetic operations
  • Number.toFixed() — Limit decimals (10) to avoid floating point precision errors
  • String.length — Validate 12-digit limit on input
