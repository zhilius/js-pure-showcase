Calculadora funcional con suma, resta, multiplicación y división. Demuestra manejo de eventos delegados, formato de números, detección de división por cero y límite de 12 dígitos.

Cómo funciona

Se usan dos variables de estado: currentValue (el número que se está escribiendo) y operator (la operación pendiente). Al presionar un dígito se concatena a currentValue. Al presionar un operador se almacena el valor actual como storedValue y se limpia currentValue. Al presionar = se ejecuta la operación almacenada sobre storedValue y currentValue.

function handleClick(e) {
  var val = e.target.textContent;
  if (!isNaN(val) || val === '.') {
    if (currentValue.length < 12) currentValue += val;
  } else if (['+', '-', '×', '÷'].indexOf(val) >= 0) {
    if (operator) currentValue = compute();
    storedValue = currentValue;
    operator = val;
    currentValue = '';
  } else if (val === '=') {
    currentValue = compute();
    operator = '';
  }
  display.textContent = currentValue || '0';
}

function compute() {
  var a = parseFloat(storedValue), b = parseFloat(currentValue);
  if (operator === '÷' && b === 0) return 'Error';
  var result = {
    '+': a + b, '-': a - b, '×': a * b, '÷': a / b
  }[operator];
  return String(parseFloat(result.toFixed(10)));
}

El botón C reinicia todos los valores. Divisiones por cero muestran «Error».

APIs de JavaScript utilizadas

  • Event delegation — Un solo click listener en el contenedor de botones con e.target.textContent
  • parseFloat() — Conversión de string a número para operaciones aritméticas
  • Number.toFixed() — Limitar decimales (10) para evitar errores de precisión flotante
  • String.length — Validar límite de 12 dígitos en la entrada
