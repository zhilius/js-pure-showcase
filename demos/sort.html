Live visualization of 5 sorting algorithms on a 60-element array: Bubble Sort, Quick Sort, Selection Sort, Insertion Sort, and Merge Sort. Each comparison produces a musical tone via Web Audio API — higher pitches for larger values. Demonstrates async/await with setTimeout to animate step by step.

How It Works

The array is initialized with values 1..60 and shuffled with Fisher-Yates. Each algorithm runs asynchronously: at each comparison it marks the elements (red), plays a tone proportional to the value, pauses via sleep(), and redraws. Sorted elements turn green.

async function bubbleSort() {
  for (var i = 0; i < arr.length; i++) {
    for (var j = 0; j < arr.length - i - 1; j++) {
      comparing = [j, j + 1]; draw();
      sortSound(arr[j] > arr[j + 1] ? arr[j + 1] : arr[j]);
      await sleep();
      if (arr[j] > arr[j + 1]) swap(arr, j, j + 1);
    }
  }
}

async function selectionSort() {
  for (var i = 0; i < N - 1; i++) {
    var minIdx = i;
    for (var j = i + 1; j < N; j++) {
      if (arr[j] < arr[minIdx]) minIdx = j;
    }
    swap(arr, i, minIdx);
  }
}

async function insertionSort() {
  for (var i = 1; i < N; i++) {
    var key = arr[i], j = i - 1;
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j]; j--;
    }
    arr[j + 1] = key;
  }
}

async function mergeSort(lo, hi) {
  if (lo >= hi) return;
  var mid = Math.floor((lo + hi) / 2);
  await mergeSort(lo, mid);
  await mergeSort(mid + 1, hi);
  await merge(lo, mid, hi);
}

Each algorithm uses the same sleep() helper (await new Promise(r => setTimeout(r, speed))) and the speed slider controls the delay between steps. A sortSound(val) function creates a short OscillatorNode tone, mapping the bar value to frequency (200–900 Hz). Toggle between algorithms at any time.

JavaScript APIs Used

  • Async/Await — async function + await to pause algorithm execution at each step
  • Promise — new Promise(resolve => setTimeout(resolve, ms)) as sleep
  • CanvasRenderingContext2D — fillRect() to draw bars with height proportional to value
  • Web Audio API (OscillatorNode + GainNode) — Generate a short tone per comparison for audible feedback
  • Fisher-Yates shuffle — O(n) algorithm to shuffle the initial array
