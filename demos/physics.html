What It Teaches

Canvas 2DClassical PhysicsPer-frame Animation

Balls with gravity, elastic bounce, friction, and specular highlights. Demonstrates step-by-step physics integration in each animation frame.

How It Works

Each ball has a position (x, y), velocity (vx, vy), radius, color, and coefficient of restitution (e). On each frame:

b.vy += grav;          // gravity
b.x += b.vx; b.y += b.vy;  // move

// Border collision
if (b.x - b.r < 0) { b.x = b.r; b.vx *= -b.e; }
if (b.y + b.r > 180) {
  b.y = 180 - b.r;
  b.vy *= -b.e;       // bounce
  b.vx *= 0.98;       // friction when touching the ground
}

The coefficient of restitution e is random between 0.7 and 0.9, simulating different materials. The specular highlight is drawn as a semi-transparent white circle offset upward and to the left of each ball's center.

JavaScript APIs Used

  • CanvasRenderingContext2D — arc() for circles, fillStyle with transparency for highlights
  • requestAnimationFrame — Physics loop synchronized with the display refresh rate
  • Math.random() — Generation of random values for radius, velocity, and restitution
  • Input range — <input type="range"> for real-time gravity control
