Demostración de dos características ES6: (1) generador infinito de Fibonacci con function* e yield, y (2) Proxy que intercepta gets y sets para registrar operaciones.

Cómo funciona

Fibonacci Generator: function* fibGen() produce la secuencia infinita de Fibonacci. Cada llamada a .next() devuelve el siguiente número. El botón «Siguiente Fibonacci» avanza el generador y muestra el resultado.

function* fibGen() {
  var a = 0, b = 1;
  while (true) {
    yield a;
    var temp = a;
    a = b;
    b = temp + b;
  }
}

var fib = fibGen();
button.addEventListener('click', function() {
  output.textContent = fib.next().value;
});

Proxy: Se crea un objeto objetivo target = {} y un handler con trampas get y set. El get registra la propiedad leída y su valor. El set registra la propiedad, el valor anterior y el nuevo. Las operaciones se muestran en un área de log.

var handler = {
  get: function (target, prop) {
    var value = target[prop];
    log('Get: ' + prop + ' = ' + (value !== undefined ? value : 'undefined'));
    return value;
  },
  set: function (target, prop, value) {
    var old = target[prop];
    log('Set: ' + prop + ' = ' + value + ' (antes: ' + (old !== undefined ? old : 'undefined') + ')');
    target[prop] = value;
    return true;
  }
};
var proxy = new Proxy(target, handler);

APIs de JavaScript utilizadas

  • function* / yield — Generador infinito que produce valores bajo demanda sin calcular la secuencia completa
  • Generator.next() — Avanza el generador al próximo valor { value, done }
  • Proxy — new Proxy(target, handler) para meta-programación
  • Handler.get / Handler.set — Trampas que interceptan acceso y asignación de propiedades
