Qué enseña

Pizarra de dibujo libre con strokeStyle dinámico, lineWidth variable y lineCap: "round". Soporta mouse y táctil, con selector de color y slider de tamaño de pincel.

Cómo funciona

En mousedown se guarda la posición actual como último punto. En mousemove, si el mouse está presionado (isDrawing), se dibuja una línea desde el último punto hasta la posición actual con ctx.beginPath(), moveTo(), lineTo(), stroke(), y se actualiza el último punto. En mouseup se desactiva el dibujo.

var lastX, lastY, isDrawing = false;

canvas.addEventListener('mousedown', function (e) {
  isDrawing = true;
  var rect = canvas.getBoundingClientRect();
  lastX = (e.clientX - rect.left) * dpr;
  lastY = (e.clientY - rect.top) * dpr;
});

canvas.addEventListener('mousemove', function (e) {
  if (!isDrawing) return;
  var rect = canvas.getBoundingClientRect();
  var curX = (e.clientX - rect.left) * dpr;
  var curY = (e.clientY - rect.top) * dpr;
  ctx.beginPath();
  ctx.moveTo(lastX, lastY);
  ctx.lineTo(curX, curY);
  ctx.stroke();
  lastX = curX; lastY = curY;
});

Los eventos táctiles usan e.preventDefault() para evitar scroll, y acceden a e.touches[0]. El color se obtiene de <input type="color"> y el tamaño del <input type="range">. El botón «Limpiar» llama a clearRect().

APIs de JavaScript utilizadas

  • CanvasRenderingContext2D — beginPath(), moveTo(), lineTo(), stroke(), clearRect()
  • MouseEvent — mousedown, mousemove, mouseup, mouseleave con clientX/clientY
  • TouchEvent — touchstart, touchmove, touchend con e.touches[0]
  • HTMLInputElement — input[type=color] y input[type=range] con listener input
