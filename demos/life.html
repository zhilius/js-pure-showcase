Implementation of Conway's classic cellular automaton with exact rules, toroidal borders, double buffer in Uint8Array, and cell toggle by click.

How It Works

Two grids (grid and next) are maintained as Uint8Array of 50×50 cells. In each generation, for each cell the 8 live neighbors are counted (with toroidal wrap using modulo). The rules are applied:

function countNeighbors(g, x, y) {
  var sum = 0;
  for (var i = -1; i <= 1; i++) {
    for (var j = -1; j <= 1; j++) {
      if (i === 0 && j === 0) continue;
      var col = (x + i + 50) % 50;
      var row = (y + j + 50) % 50;
      sum += g[col + row * w];
    }
  }
  return sum;
}

function nextGen() {
  for (var x = 0; x < w; x++) {
    for (var y = 0; y < h; y++) {
      var idx = x + y * w;
      var neighbors = countNeighbors(grid, x, y);
      if (grid[idx] === 1) {
        next[idx] = (neighbors < 2 || neighbors > 3) ? 0 : 1;
      } else {
        next[idx] = (neighbors === 3) ? 1 : 0;
      }
    }
  }
  // swap buffers
  var tmp = grid; grid = next; next = tmp;
}

The toroidal effect is achieved with (x + i + w) % w. Live cells are drawn in green, dead ones in black. A click converts a dead cell to live and vice versa.

JavaScript APIs Used

  • Uint8Array — Typed buffer for the two grids of 2500 cells each
  • CanvasRenderingContext2D — fillRect() for each cell in the grid (10×10 px)
  • requestAnimationFrame — Animation loop for the generations
  • MouseEvent — click for toggling cells with relative coordinates converted to grid indices
  • Modulo Operator — (x + i + w) % w for toroidal borders
