Implementación del clásico autómata celular de Conway con reglas exactas, bordes toroidales, doble buffer en Uint8Array y toggle de celdas por click.

Cómo funciona

Se mantienen dos grillas (grid y next) como Uint8Array de 50×50 celdas. En cada generación, para cada celda se cuentan los 8 vecinos vivos (con wrap toroidal usando módulo). Se aplican las reglas:

function countNeighbors(g, x, y) {
  var sum = 0;
  for (var i = -1; i <= 1; i++) {
    for (var j = -1; j <= 1; j++) {
      if (i === 0 && j === 0) continue;
      var col = (x + i + 50) % 50;
      var row = (y + j + 50) % 50;
      sum += g[col + row * w];
    }
  }
  return sum;
}

function nextGen() {
  for (var x = 0; x < w; x++) {
    for (var y = 0; y < h; y++) {
      var idx = x + y * w;
      var neighbors = countNeighbors(grid, x, y);
      if (grid[idx] === 1) {
        next[idx] = (neighbors < 2 || neighbors > 3) ? 0 : 1;
      } else {
        next[idx] = (neighbors === 3) ? 1 : 0;
      }
    }
  }
  // swap buffers
  var tmp = grid; grid = next; next = tmp;
}

El toroidal se logra con (x + i + w) % w. Las celdas vivas se dibujan en verde, las muertas en negro. El click convierte una celda muerta en viva y viceversa.

APIs de JavaScript utilizadas

  • Uint8Array — Buffer tipado para las dos grillas de 2500 celdas cada una
  • CanvasRenderingContext2D — fillRect() para cada celda en la grilla (10×10 px)
  • requestAnimationFrame — Loop de animación para las generaciones
  • MouseEvent — click para toggle de celdas con coordenadas relativas convertidas a índices de grilla
  • Operador módulo — (x + i + w) % w para bordes toroidales
