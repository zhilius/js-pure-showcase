Classic Snake game: control the snake with arrow keys, randomly appearing food, growth, wall and self collision, scoring, and a game over screen.

How It Works

The snake is an array of segments with (x, y) coordinates on a 20×20 grid. On each tick (every 120 ms via recursive setTimeout), the head moves in the current direction, checking for collision with food, walls, or its own body, and redraws everything.

function gameLoop() {
  if (gameOver) { drawGameOver(); return; }

  var head = { x: snake[0].x + dir.x, y: snake[0].y + dir.y };

  // Wall collision
  if (head.x < 0 || head.x >= 20 || head.y < 0 || head.y >= 20) {
    gameOver = true; drawGameOver(); return;
  }

  // Self collision
  for (var i = 0; i < snake.length; i++) {
    if (snake[i].x === head.x && snake[i].y === head.y) {
      gameOver = true; drawGameOver(); return;
    }
  }

  snake.unshift(head);
  if (head.x === food.x && head.y === food.y) {
    score++; spawnFood();
  } else snake.pop();

  draw();
  setTimeout(gameLoop, 120);
}

Food spawns at random positions not occupied by the snake (do{...}while(occupied)). Game over shows a semi-transparent overlay with the score and a restart button.

JavaScript APIs Used

  • KeyboardEvent — keydown with e.key for ArrowUp/Down/Left/Right (with reversal prevention)
  • Recursive setTimeout — Non-blocking game loop with 120 ms interval
  • CanvasRenderingContext2D — fillRect() for segments and food, fillText() for score, fillStyle with rgba() for overlay
  • Math.random() — Random food position on the grid
  • Array.unshift()/pop() — Insert new head and remove tail for movement
