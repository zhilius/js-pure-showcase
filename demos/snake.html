Juego Snake clásico: control de serpiente con flechas, comida que aparece aleatoriamente, crecimiento, colisiones con pared y auto-colisión, puntuación y pantalla de game over.

Cómo funciona

La serpiente es un arreglo de segmentos con coordenadas (x, y) en una grilla de 20×20. En cada tick (cada 120 ms via setTimeout recursivo) se mueve la cabeza en la dirección actual, se verifica colisión con comida, pared o el propio cuerpo, y se redibuja todo.

function gameLoop() {
  if (gameOver) { drawGameOver(); return; }

  var head = { x: snake[0].x + dir.x, y: snake[0].y + dir.y };

  // Colisión con pared
  if (head.x < 0 || head.x >= 20 || head.y < 0 || head.y >= 20) {
    gameOver = true; drawGameOver(); return;
  }

  // Auto-colisión
  for (var i = 0; i < snake.length; i++) {
    if (snake[i].x === head.x && snake[i].y === head.y) {
      gameOver = true; drawGameOver(); return;
    }
  }

  snake.unshift(head);
  if (head.x === food.x && head.y === food.y) {
    score++; spawnFood();
  } else snake.pop();

  draw();
  setTimeout(gameLoop, 120);
}

La comida se genera en posiciones aleatorias que no ocupen la serpiente (do{...}while(ocupado)). El game over muestra un overlay semitransparente con la puntuación y un botón de reinicio.

APIs de JavaScript utilizadas

  • KeyboardEvent — keydown con e.key para ArrowUp/Down/Left/Right (con prevención de reversión)
  • setTimeout recursivo — Loop de juego no bloqueante con intervalo de 120 ms
  • CanvasRenderingContext2D — fillRect() para segmentos y comida, fillText() para puntuación, fillStyle con rgba() para overlay
  • Math.random() — Posición aleatoria de la comida en la grilla
  • Array.unshift()/pop() — Inserción de nueva cabeza y eliminación de cola para movimiento
