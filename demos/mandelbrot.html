Renderizado píxel a píxel del conjunto de Mandelbrot con iteración de números complejos y mapeo de colores. Incluye zoom clickeable que re-renderiza con el nuevo centro y escala.

Cómo funciona

Cada píxel se mapea a un punto en el plano complejo según el centro y el zoom actual. Se itera z = z² + c hasta que la magnitud excede 2 o se alcanza el máximo de iteraciones (100). El color se calcula con Math.sqrt/Math.pow para una paleta suave. Se usa ImageData + Uint8ClampedArray para manipulación directa de píxeles.

for (var x = 0; x < w; x++) {
  for (var y = 0; y < h; y++) {
    var a = map(x, 0, w, cx - zoom, cx + zoom);
    var b = map(y, 0, h, cy - zoom, cy + zoom);
    var n = 0, za = a, zb = b;
    while (n < 100) {
      var aa = za * za - zb * zb;
      var bb = 2 * za * zb;
      za = aa + a; zb = bb + b;
      if (za * za + zb * zb > 4) break;
      n++;
    }
    var hue = Math.pow(n / 100, 0.6) * 360;
    var sat = n === 100 ? 0 : 100;
    /* colorear píxel con HSL */
  }
}

Al hacer click, se calcula el nuevo centro en coordenadas del plano complejo y se reduce el zoom a la mitad, re-renderizando el fractal completo.

APIs de JavaScript utilizadas

  • CanvasRenderingContext2D.putImageData() — Renderizado directo de píxeles desde el buffer
  • ImageData — createImageData() para buffer en memoria
  • Uint8ClampedArray — Arreglo tipado con clamping automático (0–255) para valores RGBA
  • Math — sqrt(), pow() para mapeo de color; aritmética de números complejos
  • MouseEvent — click con getBoundingClientRect() para coordenadas relativas
