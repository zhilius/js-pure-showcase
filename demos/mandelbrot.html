Pixel-by-pixel rendering of the Mandelbrot set with complex number iteration and color mapping. Includes clickable zoom that re-renders with the new center and scale.

How It Works

Each pixel is mapped to a point in the complex plane based on the current center and zoom. It iterates z = z² + c until the magnitude exceeds 2 or the maximum iterations (100) is reached. The color is calculated with Math.sqrt/Math.pow for a smooth palette. ImageData + Uint8ClampedArray is used for direct pixel manipulation.

for (var x = 0; x < w; x++) {
  for (var y = 0; y < h; y++) {
    var a = map(x, 0, w, cx - zoom, cx + zoom);
    var b = map(y, 0, h, cy - zoom, cy + zoom);
    var n = 0, za = a, zb = b;
    while (n < 100) {
      var aa = za * za - zb * zb;
      var bb = 2 * za * zb;
      za = aa + a; zb = bb + b;
      if (za * za + zb * zb > 4) break;
      n++;
    }
    var hue = Math.pow(n / 100, 0.6) * 360;
    var sat = n === 100 ? 0 : 100;
    /* color pixel with HSL */
  }
}

On click, the new center is calculated in complex plane coordinates and the zoom is reduced by half, re-rendering the entire fractal.

JavaScript APIs Used

  • CanvasRenderingContext2D.putImageData() — Direct pixel rendering from the buffer
  • ImageData — createImageData() for in-memory buffer
  • Uint8ClampedArray — Typed array with automatic clamping (0–255) for RGBA values
  • Math — sqrt(), pow() for color mapping; complex number arithmetic
  • MouseEvent — click with getBoundingClientRect() for relative coordinates
