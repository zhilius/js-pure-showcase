What It Teaches

Rain effect of katakana characters in Matrix style. Demonstrates the use of requestAnimationFrame with a semi-transparent canvas to create the trail effect.

How It Works

An array of 30 columns is defined, each with an object that has y (vertical position) and speed (velocity in pixels per frame). In each frame a semi-transparent black rectangle is painted over the entire canvas (fillStyle = "rgba(0,0,0,0.05)"), creating the effect of characters gradually fading.

var columns = 30;
var drops = [];
for (var i = 0; i < columns; i++) {
  drops[i] = { y: Math.random() * -h, speed: 1 + Math.random() * 3 };
}

function draw() {
  ctx.fillStyle = "rgba(0,0,0,0.05)";
  ctx.fillRect(0, 0, canvas.width, canvas.height);
  ctx.fillStyle = "#0f0";
  ctx.font = fontSize + "px monospace";

  for (var i = 0; i < drops.length; i++) {
    var char = String.fromCharCode(0x30A0 + Math.random() * 96);
    ctx.fillText(char, i * fontSize, drops[i].y);
    drops[i].y += drops[i].speed;
    if (drops[i].y > canvas.height) {
      drops[i].y = -fontSize;
      drops[i].speed = 1 + Math.random() * 3;
    }
  }
  requestAnimationFrame(draw);
}

Characters are randomly generated from the Katakana Unicode block (U+30A0–U+30FF). Each drop resets from the top with a random speed when it exits the screen.

JavaScript APIs Used

  • CanvasRenderingContext2D — fillText() to draw characters, semi-transparent fillRect() for the trail
  • requestAnimationFrame — Animation loop for continuous motion
  • String.fromCharCode() — Generation of random katakana Unicode characters in the range U+30A0–U+30FF
  • Math.random() — Random initial position, variable speed and random characters
