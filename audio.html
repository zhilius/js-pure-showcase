Dos modos: (1) micrófono real vía getUserMedia() conectado a un AnalyserNode que provee datos de frecuencia, y (2) sintetizador que genera 4 notas musicales con osciladores.

Cómo funciona

Modo micrófono: Obtiene el stream de audio con getUserMedia({ audio: true }), crea un AudioContext, conecta el stream a un AnalyserNode con fftSize: 128. En cada frame lee los datos de frecuencia con getByteFrequencyData() como Uint8Array y dibuja barras coloreadas con hsl() en el canvas.

var stream = await navigator.mediaDevices.getUserMedia({ audio: true });
var audioCtx = new AudioContext();
var src = audioCtx.createMediaStreamSource(stream);
var analyser = audioCtx.createAnalyser();
analyser.fftSize = 128;
src.connect(analyser);
dataArr = new Uint8Array(analyser.frequencyBinCount);

function loop() {
  analyser.getByteFrequencyData(dataArr);
  /* dibujar barras según dataArr[i] / 255 * 140 */
  requestAnimationFrame(loop);
}

Modo sintetizador: Crea 4 osciladores con frecuencias de Do, Mi, Sol, Do (261.63, 329.63, 392, 523.25 Hz). Cada uno tiene un GainNode con ataque inmediato y decaimiento exponencial en 0.4 segundos, creando un arpegio.

APIs de JavaScript utilizadas

  • MediaDevices.getUserMedia() — Acceso al micrófono del usuario
  • AudioContext — Contexto de audio para procesar y generar sonido
  • AnalyserNode — Análisis de frecuencia en tiempo real con FFT
  • OscillatorNode — Generación de tonos senoidales con frecuencia configurable
  • GainNode — Control de volumen con setValueAtTime y exponentialRampToValueAtTime
  • Uint8Array — Buffer tipado para datos de frecuencia del analizador
